# CarCrafters

CarCrafters is a premium car dealership and rental web application built with Next.js, Tailwind CSS, and Framer Motion. It features a modern, dynamic design with smooth animations and an interactive user experience.

## Features

- **Dynamic Navigation**: Single-page application style navigation without page reloads.
- **Hero Section**: Immersive video background with staggered entrance animations.
- **Featured Inventory**: Interactive car cards with hover effects and details.
- **Trusted Brands**: Auto-scrolling slider showcasing partner brands.
- **About & Why Us**: Scroll-triggered animations highlighting company stats and values.
- **Call to Action**: Engaging CTA section with animated elements.
- **Responsive Design**: Fully optimized for all screen sizes.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Sliders**: [Swiper](https://swiperjs.com/)
- **Icons/Fonts**: Custom fonts (Orbitron, Open Sans)

## Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/car-crafters.git
    cd car-crafters
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.

## Project Structure

- `app/`: Main application code (Next.js App Router).
  - `components/`: Reusable UI components (Navbar, Hero, About, etc.).
  - `page.tsx`: Main entry point with state-based navigation.
  - `layout.tsx`: Root layout definition.
  - `globals.css`: Global styles and Tailwind configuration.
- `public/`: Static assets (images, videos, fonts).
